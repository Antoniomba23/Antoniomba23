### 💼 Proyectos Destacados

🔹 **[StudyFiles](https://github.com/Antoniomba23/PROYECTO-FINAL-ANDROID-SEGUNDO-DAM)**  
Aplicación Android para la compartición de apuntes y recursos académicos. Desarrollada en Java con Room/SQLite, backend en la nube con Supabase, Retrofit para API REST y Gemini API para inteligencia artificial. *Proyecto Final de Ciclo (DAM).*

🔹 **[eGROOMIE](https://egroomie.com)**  
Plataforma integral web y móvil para digitalizar hoteles, restaurantes y ocio en Guinea Ecuatorial. Incluye sistemas de reserva, registro de negocios y testing activo en Google Play. Desarrollada con Next.js, Firebase y React.

🔹 **[GuineaVentas](https://guineaventas.netlify.app)**  
Plataforma de comercio electrónico para Guinea Ecuatorial que conecta vendedores locales con compradores. Facilita transacciones y pedidos rápidos con un catálogo dinámico y responsivo.

🔹 **[NOA - Asistente Virtual](https://github.com/Antoniomba23/)**  
Asistente conversacional de WhatsApp Business enfocado en la automatización de atención al cliente, utilizando APIs de Inteligencia Artificial para respuestas contextuales inteligentes.

🔹 **[API REST Multihilo](https://github.com/Antoniomba23/Gestor_Pedidos)**  
API robusta para gestión de tareas construida con Spring Boot y MongoDB, implementando programación concurrente multihilo (PSP) y endpoints completamente testeados.
